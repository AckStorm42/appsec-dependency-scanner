# Security Policy

## 🛡️ Supported Versions

This project is a learning and demonstration tool. Only the latest version on the `master` branch is considered actively maintained.

---

## 🚨 Reporting a Vulnerability

If you discover a security issue in this project, please report it responsibly.

### How to report

* Open a **GitHub issue** (preferred for non-sensitive findings)
* For sensitive issues, contact the maintainer directly via GitHub

Please include:

* Description of the issue
* Steps to reproduce
* Potential impact
* Suggested fix (if available)

---

## 🔍 Scope

This project:

* Does **not** handle sensitive user data
* Does **not** include authentication or user accounts
* Is intended for local or controlled environment use

---

## ⚠️ Disclaimer

This tool is provided **for educational and informational purposes only**.

* It does **not guarantee complete vulnerability detection**
* It should **not be relied upon as a sole security control**
* Always use established security tools and practices in production environments

---

## 🔐 Best Practices for Users

When using this tool:

* Do not commit API keys or secrets to source control
* Use environment variables for sensitive values (e.g., `NVD_API_KEY`)
* Run dependency scans as part of a broader security process
